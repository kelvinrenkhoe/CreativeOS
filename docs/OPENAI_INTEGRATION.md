# OpenAI Integration

OpenAI is the first AI provider for CreativeOS.

CreativeOS will use OpenAI to help generate:

- AI prompts
- Social media captions
- Weekly content plans
- Campaign ideas
- Song story concepts

OpenAI is an integration, not the whole product.
CreativeOS remains a workflow toolkit first.

## Environment Variable

Do not store API keys in the repository.

Use:

export OPENAI_API_KEY="your_api_key_here"
